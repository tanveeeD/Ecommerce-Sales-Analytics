# E-Commerce Sales Analytics Dashboard

## Overview

This project presents an end-to-end sales analytics solution using Excel, SQL, Python, and Power BI. The objective is to analyze retail sales data, identify business trends, and build an interactive dashboard for decision-making.

---

## Tools & Technologies

- Excel
- SQL
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI

---

## Project Workflow

- Data Cleaning and Preparation
- SQL-based Business Analysis
- Exploratory Data Analysis (EDA) using Python
- Interactive Power BI Dashboard Development
- Business Insights Generation

---

## Key Features

- Sales and Profit Analysis
- Monthly Sales Trend
- Category-wise Performance
- Regional Sales Analysis
- KPI Dashboard
- Interactive Visualizations

## Business Insights

- Technology category generated the highest sales.
